About Music.com
Music.com is website provides Entertainment to user with latest and well organized music tracks. User can find almost any music track on Music.com. It also provides search function to user. Specialities of Music.com is to provide music player interface within web page where user can play latest music tracks, listen to different playlists and albums and user can also create his own playlists. User’s playlists can also be shown to other users.
This Website Include following page
1. Explore: Provide list of Latest Tracks and Latest playlists created by other users. All tracks are divided based on categories like Party, Romance, Retro, Rock etc.
2. Trending: List all popular Tracks and Albums.
3. Music Library: List of all Albums/Playlist arranged by alphabetical letters. It provide list of all tracks available on Music.com. User can also have search function in this page.
4. My Lists: Provide Playlist created by current user. User can play list by simply clicking on Play all button. User can create new playlist there.
5. Login/Register: Provide interface to user for Registration and Login.
6. About Music.com: Contain description about Music.com. It answers questions like “What is Music.com?”, “Why Music.com?”, “How to?” etc...
7. Contact Us: Contact details of Music.com. It also contains simple form for user to fill up and send message to Admin of Music.com.
